# AABB Tree
Simple implementation of an AABB Tree (Axis Aligned Bounding Box Tree) to optimize 3d collision detection

## Commands
**Run Test**: `npm test`  
**Generate the docs**: `npm docs`  
**Run the example**: `npm example`  
**Build the package**: `npm build`
